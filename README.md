# Автосалон «Врум-Бум»: нормализация данных и аналитика продаж


[![PostgreSQL](https://img.shields.io/badge/PostgreSQL-12+-blue)](https://www.postgresql.org/)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

Проект по нормализации и структурированию данных для сети автосалонов «Врум-Бум». Реализована миграция сырых данных в PostgreSQL, создана нормализованная схема БД (до 3NF) и набор аналитических SQL‑запросов.

## Описание проекта

Сеть автосалонов «Врум-Бум» столкнулась с проблемой неструктурированных данных о продажах автомобилей (2015–2023 гг.). Проект решает задачу:
* нормализации и структурирования данных;
* переноса в реляционную БД PostgreSQL;
* реализации аналитических запросов для бизнес‑аналитики.

**Цель проекта:** продемонстрировать навыки работы с:
* нормализацией данных (до 3НФ);
* проектированием схемы БД;
* написанием сложных SQL‑запросов для аналитики;
* обработкой и миграцией данных из CSV в PostgreSQL.

## Функциональность и задачи

### Этап 1. Создание и заполнение БД

* **Схема `raw_data`:** таблица `sales` для хранения исходных данных из CSV.
* **Схема `car_shop`:** нормализованные таблицы (3НФ) с первичными и внешними ключами, автоинкрементом, ограничениями (`NOT NULL`, `CONSTRAINTS`).
* **Миграция данных:** перенос из `raw_data.sales` в нормализованные таблицы с помощью `INSERT INTO … SELECT …`.
* **Обоснование модели:** файл `data_model_explanation.md` с описанием выбора структуры и типов данных.

### Этап 2. Аналитические запросы

Реализованы 6 SQL‑запросов для бизнес‑аналитики:
1. Процент моделей без `gasoline_consumption`.
2. Средняя цена автомобилей по брендам и годам (с учётом скидки).
3. Средняя цена по месяцам в 2022 году (с учётом скидки).
4. Список купленных машин у каждого пользователя (`STRING_AGG`).
5. Макс./мин. цена по странам происхождения (без учёта скидки).
6. Количество пользователей из США (по коду телефона `+1`).

## Использование и запуск

### Предварительные требования

* Установленный PostgreSQL (версия 12+).

* Исходный CSV‑файл с данными о продажах (см. раздел «Структура проекта»).
